Keyboard Shortcuts

This reader has a few useful shortcut keys that you might want to know:

  • D — Toggle Dark Mode
  • Left Arrow, K — Previous Page
  • Right Arrow, J — Next Page

Some search filters are also available:

  • chapter — Restrict search to a certain chapter. Usage example: protestant chapter:16 would search for all instances of "protestant" in chapter 16.

Other good things to know:

  • The search bar is sticky, so you can use it anywhere.
  • At the end of each chapter, in the Second Thoughts section, the terms of significance are clickable, and will take you to a search of that term in the chapter.
